# mazha Tap — Frontend

Next.js frontend for the mazha Tap rain-tapping decision tool.

## Run locally

```bash
npm install
npm run dev
```

The app will be available at http://localhost:3000.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Leaflet + React-Leaflet
- next-themes
- lucide-react
